Как запускать на экзамене
Один раз из корня:

npm install
Перед каждым новым заданием — останови предыдущий сервер (Ctrl+C), иначе будет ошибка EADDRINUSE (порт 3000 занят).

Группа	Команда
16–32
node 16/16_http_server_404.js и т.д.
33–37
node 33/33_websocket_browser.js (нужен ws)
38–45
Нужны запущенные БД (PostgreSQL, MongoDB, Redis)
46–53
node 46/46_express_middleware_routes.js и т.д.
54
cd 54 → npm install → npm run start:dev
55
cd 55 → npm install → npm run start:dev
Все команды есть в запуск.txt.

Установка и запуск баз данных
PostgreSQL (для 38-41, 45):
# Windows (CMD как администратор)
net start postgresql-x64-15

# Или через PowerShell
Start-Service postgresql

# Создание базы данных
createdb testdb

# Подключение
psql -U postgres -d testdb

# macOS
brew install postgresql
brew services start postgresql
createdb testdb

# Linux
sudo service postgresql start
sudo -u postgres createdb testdb


MongoDB (42):
# Windows - создайте папку для данных
mkdir C:\data\db

# Запуск MongoDB
mongod --dbpath C:\data\db

# Или через службу
net start MongoDB

# macOS
brew install mongodb-community
brew services start mongodb-community

# Linux
sudo systemctl start mongod


Redis (44):
# Windows (скачайте с https://github.com/microsoftarchive/redis/releases)
redis-server

brew install redis

# Linux
